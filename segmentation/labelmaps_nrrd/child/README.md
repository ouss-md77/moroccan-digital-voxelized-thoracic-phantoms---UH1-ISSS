# Child Voxelized NRRD Labelmaps

This folder describes the anonymized voxelized thoracic labelmaps for the Moroccan child group.

The NRRD files are provided as a ZIP file in the GitHub Releases section because the files are large.

## Files included in the ZIP archive

* E1.nrrd
* E2.nrrd
* E3.nrrd
* E4.nrrd
* E5.nrrd
* E6.nrrd
* E7.nrrd
* E8.nrrd
* E9.nrrd
* E10.nrrd

## Label values

| Label | Structure |
| ----- | --------- |
| 1     | Lungs     |
| 2     | Body      |
| 3     | Bones     |

## Data availability

The anonymized child labelmaps are available in the release file:

`child_labelmaps_nrrd.zip`

## Privacy note

Only anonymized voxelized labelmaps are shared.

Original CT images, DICOM files, patient names, hospital identifiers, and private metadata are not included.
