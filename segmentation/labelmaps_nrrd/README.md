# Voxelized NRRD Labelmaps

This folder contains anonymized voxelized thoracic labelmaps in NRRD format.

The data are organized into three groups:

* `male/` for Moroccan adult male labelmaps
* `female/` for Moroccan adult female labelmaps
* `child/` for Moroccan child labelmaps

## Label values

For male and child phantoms:

| Label | Structure |
| ----- | --------- |
| 1     | Lungs     |
| 2     | Body      |
| 3     | Bones     |

For female phantoms:

| Label | Structure |
| ----- | --------- |
| 1     | Lungs     |
| 2     | Body      |
| 3     | Breasts   |
| 4     | Bones     |

## Privacy note

Only anonymized voxelized labelmaps are shared.

Original CT images, DICOM files, hospital identifiers, patient names, and private metadata are not included in this repository.

These files are provided for academic and research purposes only.
