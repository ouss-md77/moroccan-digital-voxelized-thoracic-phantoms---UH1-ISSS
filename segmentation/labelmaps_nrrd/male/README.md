# Male Voxelized NRRD Labelmaps

This folder contains anonymized voxelized thoracic labelmaps for the Moroccan adult male group.

## Files

The labelmaps are named using anonymized patient codes:

* H1.nrrd
* H2.nrrd
* H3.nrrd
* H4.nrrd
* H5.nrrd
* H6.nrrd
* H7.nrrd
* H8.nrrd
* H9.nrrd
* H10.nrrd

## Label values

| Label | Structure |
| ----- | --------- |
| 1     | Lungs     |
| 2     | Body      |
| 3     | Bones     |

## Privacy note

Only anonymized voxelized labelmaps are included.

Original CT images, DICOM files, patient names, hospital identifiers, and private metadata are not included.
