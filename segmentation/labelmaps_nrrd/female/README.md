# Female Voxelized NRRD Labelmaps

This folder describes the anonymized voxelized thoracic labelmaps for the Moroccan adult female group.

The NRRD files are provided as a ZIP file in the GitHub Releases section because the files are large.

## Files included in the ZIP archive

* F1.nrrd
* F2.nrrd
* F3.nrrd
* F4.nrrd
* F5.nrrd
* F6.nrrd
* F7.nrrd
* F8.nrrd
* F9.nrrd
* F10.nrrd

## Label values

| Label | Structure |
| ----- | --------- |
| 1     | Lungs     |
| 2     | Body      |
| 3     | Breasts   |
| 4     | Bones     |

## Data availability

The anonymized female labelmaps are available in the release file:

`female_labelmaps_nrrd.zip`

## Privacy note

Only anonymized voxelized labelmaps are shared.

Original CT images, DICOM files, patient names, hospital identifiers, and private metadata are not included.
