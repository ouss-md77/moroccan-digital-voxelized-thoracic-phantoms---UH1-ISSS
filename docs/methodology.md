# Methodology

This project followed a workflow based on CT image processing, segmentation, morphometric analysis, and representative phantom selection.

The main steps were:

1. Importing thoracic CT images into 3D Slicer
2. Automatic segmentation using AI-based tools
3. Manual correction of segmented structures
4. Exporting voxelized labelmaps
5. Calculating lung volume, Chest Wall Thickness, and Anteroposterior Diameter
6. Comparing segmentation methods
7. Selecting representative phantoms using a multicriteria score
