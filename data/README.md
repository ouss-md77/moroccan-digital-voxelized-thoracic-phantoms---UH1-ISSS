# Voxelized Labelmap Data

This folder contains anonymized voxelized thoracic labelmaps in NRRD format.

The data are organized into three groups:

* `H/` for Moroccan adult male labelmaps
* `F/` for Moroccan adult female labelmaps
* `E/` for Moroccan child labelmaps

## File naming

The files are named using anonymized patient codes:

* H1 to H10: male group
* F1 to F10: female group
* E1 to E10: child group

## Label values

For male and child labelmaps:

| Label | Structure |
| ----- | --------- |
| 1     | Lungs     |
| 2     | Body      |
| 3     | Bones     |

For female labelmaps:

| Label | Structure |
| ----- | --------- |
| 1     | Lungs     |
| 2     | Body      |
| 3     | Breasts   |
| 4     | Bones     |

## Privacy note

Only anonymized voxelized labelmaps are shared.

Original CT images, DICOM files, patient names, hospital identifiers, and private metadata are not included.

These data are provided for academic and research purposes only.
